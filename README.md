# my-vbot

Vbot 运行例子

## 环境要求

1. PHP >= 7.0
2. PHP fileinfo 拓展 储存文件需要用到
3. PHP gd 拓展 控制台显示二维码
4. PHP SimpleXML 拓展 解析XML

本项目运行步骤

```
$ git clone https://github.com/96qbhy/my-vbot.git
$ cd my-vbot
$ php composer.phar install
$ php run.php
```
